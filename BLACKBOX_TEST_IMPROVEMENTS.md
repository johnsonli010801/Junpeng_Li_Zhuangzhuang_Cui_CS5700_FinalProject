## 🐛 发现并修复的真实Bug

### Bug #1: XSS防护不完整 ✅ 已修复
**问题：** 用户名中的HTML标签和恶意脚本未被清理
```
输入: <script>alert("XSS")</script>
结果: 未清理，直接存储
```

**修复：** 增强 `sanitizeInput()` 函数
- 移除所有HTML标签 `<[^>]*>`
- 移除 `javascript:` 协议
- 移除 `on事件=` 处理器
- 移除危险字符 `<>'"` 
- 移除控制字符

**文件：** `backend/src/security.js`

---

### Bug #2: 并发注册竞态条件 ✅ 已修复  
**问题：** 10个并发请求注册相同邮箱，全部成功
```
10个并发注册 test@test.com
结果: 10个用户全部创建成功（数据重复）
```

**修复：**
1. 邮箱标准化（toLowerCase + trim）
2. 注册前二次检查
3. 添加persist队列锁机制

**文件：** `backend/src/server.js`, `backend/src/db.js`

---

### Bug #3: 私聊创建逻辑不清晰 ✅ 已修复
**问题：** `isGroup:false` 但 `memberIds:[]` 返回不友好的错误

**修复：**
- 添加明确的前置检查
- 返回清晰的错误消息："私聊需要指定对方用户ID"
- 添加会话名称XSS清理

**文件：** `backend/src/server.js`

---

### Bug #4: 空消息和超长消息处理不当 ✅ 已修复
**问题：** 
- 空消息可以发送

**修复：**

- 正确返回错误给客户端

**文件：** `backend/src/server.js`

### Bug #5: 文件上传错误处理不足 ✅ 已修复
